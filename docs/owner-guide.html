Welcome

What this guide covers

This document walks you through everything you need to run the Durham Stick Makers website day to day. It covers how visitors experience the site, how to sign in to the admin area, and step-by-step instructions for every task you will regularly need to do.

The site was built so that any member of the committee can manage it with no technical knowledge. If you can fill in a form and drag a photograph, you can run the site.

One rule of thumb: nothing goes live to the public until you explicitly click Publish. You can create drafts, upload images, and experiment freely. Delete or unpublish anything at any time.

The public site

What visitors see

The public website has eight main areas, all linked from the top navigation bar. Every page pulls its content from the admin panel, so what you change behind the scenes appears on the site immediately.

Getting started

Signing in to admin

The admin area lives at /admin on your website. Go to your site URL followed by /admin/login and sign in with the email address and password that were set up for you.

  1. Open your website in a browser and add /admin/login to the end of the URL.
  2. Enter your email and password, then click Sign in.
  3. You will land on the Dashboard - a quick overview showing published products, drafts, pending orders, and upcoming workshops.
Tip: Bookmark the admin login page so you can get back to it quickly. If you forget your password, your developer can reset it through the Supabase dashboard.

The most important task

Adding a stick to the shop

This is the job you will do most often. Each stick is a one-of-a-kind listing with its own photos, description, price, and maker credit.

  1. Go to Products in the sidebar, then click + New listing.
  2. Enter a title (e.g. "Ram horn shepherds crook on hazel").
  3. Choose the product type and pick a category from the dropdown.
  4. Set the price in pounds.
  5. Upload photos - drag images into the upload zone or click to select files. Upload at least three: full-length, handle close-up, and collar or ferrule. The first image becomes the thumbnail. Drag to reorder.
  6. Write a short description about the stick's character, materials, and finish.
  7. Fill in the materials - handle, shank, and collar. The fields offer autocomplete.
  8. Select a maker from the dropdown so the stick is credited on the site.
  9. Tick Feature on homepage if you want this stick in the featured row.
  10. Click Save as draft to come back later, or Publish to put it live.
When you publish, the site automatically creates the payment entry in Stripe so the stick can be bought online. If you edit the price later, a new payment entry is created. You do not need to touch Stripe directly.

Editing and archiving

Click any product in the list to edit it. You can change the status between Draft, Published, and Sold at any time. To remove a product permanently, click Archive at the bottom of the edit page.

Events

Scheduling a workshop

Beyond the regular Monday and Tuesday drop-in sessions (configured in Settings), you can schedule one-off workshops, taster days, and courses.

  1. Go to Workshops in the sidebar, then click + New workshop.
  2. Enter a title, date, start time, and end time.
  3. Set the capacity and price (enter 0 for a free session).
  4. Add a description and optionally a featured image.
  5. Click Save workshop. It appears on the public Workshops page immediately.
Viewing bookings: When visitors book, their details appear at the bottom of the workshop edit page - name, email, phone, and how many places. If it fills up, the status changes to "Fully booked" automatically.

Content

Writing a blog post

The journal uses a modular block editor. Instead of one big text box, you build your post from blocks - each one a different type of content.

Block types available

Paragraph

Regular body text.

Heading

Large section heading.

Quote

Pulled quote with attribution.

Image

Inline photo with caption.

Subheading

Smaller heading.

List

Bulleted or numbered.

Divider

Visual section break.

Call to action

Button card with link.

  1. Go to Blog then + New post. Enter a title and excerpt.
  2. Click + Add block and choose a type. Use arrows to reorder. Add as many as you like.
  3. Add a featured image, set an author and category.
  4. Click Save as draft or Publish.

People

Managing members

Members appear on the About page and in the "Made by" dropdown on products.

  1. Go to Members, click + Add member.
  2. Enter name, a short bio, and upload a portrait photo.
  3. Set display order (lower = first on the About page). Leave Active ticked.
  4. Click Save member.

Communication

Reading messages

When somebody fills in the contact form, their message appears in Messages. Unread messages have a brass dot. Click to expand - it auto-marks as read. Click Reply by email to open your mail client with the address and subject pre-filled.

Photography

Tips for good photographs

Natural light

A window or shade beats a flash every time.

Three angles

Full length, handle close-up, collar detail.

Clean backgrounds

A plain surface or bench. No clutter.

Phone is fine

Modern phone camera is plenty. Wipe the lens.

Quick reference

At a glance

To add a stick to the shop

Products → + New listing → fill in details → upload photos → Publish

To schedule a workshop

Workshops → + New workshop → set date, time, capacity, price → Save

To write a blog post

Blog → + New post → title, excerpt → add blocks → Publish

To add a member

Members → + Add member → name, bio, photo → Save

To read a contact message

Messages → click the message → Reply by email

To change the opening hours

Settings → Regular sessions → edit days and times → Save settings

To update the homepage headline

Settings → Hero headline → type new text → Save settings

To feature a stick on the homepage

Products → edit the stick → tick "Feature on homepage" → Save

To mark a stick as sold

Products → edit the stick → change status to Sold (or it happens automatically via Stripe checkout)

To cancel a workshop

Workshops → edit the workshop → click "Cancel workshop" at the bottom

If something goes wrong: check the admin panel first - most issues are a toggle or a status change away. For data questions, check the Supabase dashboard. For anything else, contact your developer.
